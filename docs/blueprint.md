# **App Name**: FSO-LOS Analyzer

## Core Features:

- Input Form: Form inputs for Point A and B coordinates, tower heights, and clearance threshold.
- Results Display: Display results (LOS possible/not possible), distance, minimum clearance, and additional height needed.
- Interactive Map: A map with draggable markers for Points A and B, updating the input form. The path between the points changes color to visualize analysis.
- Elevation Profile: Interactive chart for elevation profile, line-of-sight, and critical points, clarifying axes and hover tooltips.
- LOS Calculation: Calculation of LOS using elevation data, tower heights, Earth's curvature, and clearance thresholds to provide analysis.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and reliability, essential for engineering tools.
- Background color: Light gray (#F5F5F5) providing a clean and neutral backdrop for data visualization and readability.
- Accent color: Soft Green (#8BC34A) as a positive indicator when LOS is possible.
- Use system fonts for broad compatibility and fast rendering.
- Utilize minimalist icons for map markers, input fields, and analysis indicators, focusing on clarity.
- Responsive layout adapting to screen sizes, ensuring usability on mobile devices by stacking the components.
